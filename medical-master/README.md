# Medical Shop

Quản Lý nhà thuốc

## cài đặt

### môt trường

- cài đặt nodejs LTS
- cài đặt môngdb
- chạy mongodb 

### 1 clone repo

```shell
    git clone https://github.com/nguyenhao120691/medical.git
    cd medical
```

### 2 Install libs
trong thư mục ``web``

```shell
    npm i

    npm run dev
```

trong thư mục ``api``

đầu tiên chạy mogodb 

```

```shell
    npm i -g sails 
    npm i
    sails lift
```
